# m-indica-pra-mim

Base limpa de projeto Expo com Expo Router.

## Comandos

- `npm install`
- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run lint`

## Estrutura inicial

- `app/_layout.tsx`: layout raiz com stack navigator.
- `app/index.tsx`: tela inicial simples para comecar o desenvolvimento.
